Page Secrète

Guide des Fonctionnalités Cachées

Cette page secrète regroupe toutes les fonctionnalités cachées du site de la chasse au trésor du Lycée Jean Prévost. Elle n'est pas accessible depuis le menu principal et est destinée aux administrateurs et aux enseignants.

Note importante : Ne partagez pas cette page avec les élèves ! Elle contient des informations qui pourraient compromettre l'expérience de la chasse au trésor.

Mode Hackeur

Le mode hackeur est un thème alternatif qui transforme l'apparence du site en interface de hacker, avec des couleurs vertes sur fond noir, des effets de glitch et des animations spéciales.

  • Activation du Mode Hackeur

    Il existe plusieurs façons d'activer le mode hackeur :

    Méthode 1 : Phrases secrètes

    Tapez l'une des phrases secrètes suivantes dans n'importe quel champ de texte du site :

    • je suis un hacker
    • mode matrix
    • hack the planet
    • mr robot
    • hello world
    • neo
    • 42
    • ... et bien d'autres (voir le fichier js/secret-phrases.js pour la liste complète)

    Méthode 2 : Code Konami

    Utilisez la séquence de touches du code Konami :

    ↑ ↑ ↓ ↓ ← → ← → B A

    Méthode 3 : LocalStorage

    Modifiez directement le localStorage en utilisant la console du navigateur :

    localStorage.setItem('hackerTheme', 'enabled');

  • Terminal Hackeur

    Le mode hackeur inclut un terminal interactif qui permet d'exécuter des commandes simulées.

    Comment y accéder

    Méthode 1 : Appuyez sur Ctrl + Alt + T lorsque le mode hackeur est activé.

    Méthode 2 : Cliquez sur le bouton "Ouvrir le Terminal" dans la notification du mode hackeur.

    Commandes disponibles

    • help - Affiche la liste des commandes disponibles
    • clear - Efface l'écran du terminal
    • binary - Lance le jeu de calcul binaire
    • matrix - Déclenche l'effet Matrix sur la page
    • glitch - Déclenche l'effet de distorsion
    • clue - Affiche un indice caché
    • about - À propos du terminal
    • exit - Ferme le terminal

  • Effet Matrix

    Un effet de "pluie de code" inspiré du film Matrix peut être activé sur le site.

    Comment l'activer

    Méthode 1 : Tapez la commande matrix dans le terminal hackeur.

    Méthode 2 : Utilisez la fonction JavaScript suivante dans la console du navigateur :

    toggleMatrixRain();

Menu Ressources+

Le site contient un sous-menu déroulant "Ressources+" qui regroupe plusieurs pages complémentaires.

  • Page de Cryptographie

    Une page éducative sur la cryptographie et la cybersécurité, avec des outils interactifs pour expérimenter différentes méthodes de chiffrement.

    Comment y accéder

    Via le menu : Ressources+ > Cryptographie

    URL directe : crypto.html

  • Page d'Archives

    Une page présentant des documents historiques et des photos avec une galerie filtrable et une lightbox.

    Comment y accéder

    Via le menu : Ressources+ > Archives

    URL directe : archives.html

  • Page des Ressources

    Une page regroupant des ressources pédagogiques supplémentaires.

    Comment y accéder

    Via le menu : Ressources+ > Ressources

    URL directe : ressources.html

Fonctionnalités d'Accessibilité

Le site inclut plusieurs fonctionnalités d'accessibilité pour améliorer l'expérience des utilisateurs ayant des besoins spécifiques.

  • Mode Contraste Élevé

    Un mode qui augmente le contraste des couleurs pour les utilisateurs malvoyants.

    Comment l'activer

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    toggleHighContrast();

  • Mode Texte Agrandi

    Un mode qui augmente la taille du texte pour les utilisateurs malvoyants.

    Comment l'activer

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    toggleLargeText();

  • Mode Dyslexie

    Un mode qui utilise une police de caractères spéciale pour les utilisateurs dyslexiques.

    Comment l'activer

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    toggleDyslexiaFont();

Outils de Développement

Plusieurs outils sont disponibles pour les développeurs et les administrateurs du site.

  • Réinitialisation des Données

    Permet de réinitialiser toutes les données stockées localement (progression, préférences, etc.).

    Comment l'utiliser

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    resetAllData();

  • Mode Débogage

    Active le mode débogage qui affiche des informations supplémentaires dans la console.

    Comment l'activer

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    localStorage.setItem('debugMode', 'enabled');

  • Déblocage de Tous les Mini-Jeux

    Débloque tous les mini-jeux sans avoir à scanner les QR codes.

    Comment l'utiliser

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    unlockAllMiniGames();

  • Déblocage de Toutes les Visites Virtuelles

    Débloque toutes les visites virtuelles.

    Comment l'utiliser

    Utilisez la fonction JavaScript suivante dans la console du navigateur :

    unlockAllVirtualVisits();

Easter Eggs

Le site contient plusieurs easter eggs cachés pour les utilisateurs curieux.

  • Citations Cachées

    Des citations cachées apparaissent aléatoirement dans certaines pages du site.

    Comment les trouver

    Survolez les éléments du site avec votre souris pendant quelques secondes.

  • Animation Secrète

    Une animation secrète peut être déclenchée en cliquant rapidement sur le logo du site.

    Comment l'activer

    Cliquez 10 fois rapidement sur le logo du Lycée Jean Prévost dans l'en-tête.

  • Message Caché dans le Code Source

    Un message caché est présent dans le code source de la page d'accueil.

    Comment le trouver

    Inspectez le code source de la page d'accueil (index.html) et cherchez les commentaires HTML.
